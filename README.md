# Tech Stack
   - PL   : React Native (0.79.3)
   - UI   : Nativewind (4.1.23), React Native Paper (5.14.5)
   - DB   : SQLite (15.2.12)


# Setup Project

1. Clone Project
   
   ```bash
   git clone https://github.com/fericopasaribu/Mahasiswa.git
   ```
   
3. Install Dependencies

   ```bash
   npm i
   ```

4. Install App

   Android 
   ```bash
   npm run android
   ```

    iOS 
   ```bash
   npm run ios
   ```
